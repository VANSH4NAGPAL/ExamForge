# 🤖 ServiceNow – Plugins, Now Assist & Collaboration Tools (Study Notes)

---

## 1. Plugins

- Administrators can extend ServiceNow functionality via **plugins**
- Enable: analytics, automation, integrations, and more
- Tailored to meet **company-specific needs**
- Streamline processes and optimize workflows

---

## 2. Now Assist

- AI-driven tool integrated across the ServiceNow instance
- Provides **contextual help** to users
- **Automates routine tasks**
- Guides users through processes
- Reduces friction & boosts platform adoption

### Now Assist Admin Console:
- Gateway to managing **generative AI-powered features**
- Install **product plugins** tailored to specific personas:
  - 💻 IT use cases
  - 👤 Customer use cases
  - 👥 Employee use cases
- Access via developer instance → Now Assist Admin Console

---

## 3. Sidebar Discussions

- Centralized space for **team collaboration** on records
- Helps resolve issues faster & improve outcome quality
- Conversations are **organized and easily accessible**

### How to use:
- Select **Sidebar icon** or navigate to the section to open dialog
- Use **Pin menu** to permanently pin Sidebar to screen side
- Supported across **multiple workspaces**

### Key behaviour:
- When a Sidebar discussion is created → a **tile is automatically added** to the **activity stream** of the associated task-based record
- Users can **post messages from discussion → activity stream** as "Sidebar-posted messages" (editable — can add or remove)
- ⚠️ Customers in **non-fulfiller roles** cannot access the record or conversation tile (same as work notes restriction)

---

## 4. Docked Chat Windows

Enhances collaboration with these features:

| Feature | Description |
|---------|-------------|
| **Favorites** | Mark discussions as favorite → shown under Favorites tab |
| **Information Panel** | About submenu, Participants submenu, Leave discussion button |
| **Collapse** | Minimize to tab at bottom of screen (multitasking) |
| **Close** | Close window entirely when done |

### User Presence Indicators:
| Dot Color | Meaning |
|-----------|---------|
| 🟢 **Green** | User is **logged in** |
| ⚫ **No dot** | User is **not logged in** |
| 🟠 **Orange** | User **recently logged out** |
